Team Contributions

Hardware Development

Sophia and Alex collaborated on designing the physical setup for the transmittance measurement system:

  • Selection and calibration of high-precision DS18B20 temperature sensors
  • Design of the sensor mounting system to ensure proper contact with test materials
  • Integration with Raspberry Pi 4 using GPIO pins for data collection
  • Creation of a stable test environment to minimize external temperature influences

Software Implementation

Marcus led the software development with assistance from Amira:

  • Data acquisition system to collect temperature readings at one-second intervals
  • Statistical analysis to determine when temperature stability is achieved
  • Transmittance calculation algorithms based on established building physics principles
  • Real-time monitoring system with alert capabilities for test completion

Web Interface

Amira created the web interface with input from the entire team:

  • Responsive design that works on desktop and mobile devices
  • Real-time data visualization using interactive graphs
  • Historical data access and export capabilities
  • Project documentation and educational resources

Materials Testing

Lucas and Emma focused on the experimental aspects of the project:

  • Selection of diverse building materials for testing
  • Preparation of standard-sized test samples
  • Design of testing protocols to ensure consistent results
  • Validation of results against published thermal properties
